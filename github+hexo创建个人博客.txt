环境：
Windows 10+git+node.js

验证安装
C:\Users\hqj>git --version

C:\Users\hqj>node -v

C:\Users\hqj>npm -v


hexo初始化：（创建一个存放blog的文件）
$ hexo init
$ npm install
$ hexo clean
$ hexo g
$ hexo s
访问： http://localhost:4000/ 看是否成功

生成ssh密钥
$ ssh-keygen -t rsa -C "github邮箱"
$ eval `ssh-agent -s`
$ ssh-add

github添加密钥
setting中设置


验证ssh连接
$ ssh -T git@github.com

配置Deployment
# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repository: https://github.com/gethujiang/gethujiang.github.io.git（站点：clone or download下面的地址）
  branch: master

安装插件
D:\myblog>npm install hexo-deployer-git --save


生成和部署
D:\myblog>hexo d -g
